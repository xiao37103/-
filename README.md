# Smart Charging Pile Analysis System

## Project Overview
This project is a comprehensive smart charging pile deployment analysis and visualization system integrating data collection, data processing, machine learning, backend development, and frontend visualization.

The system focuses on charging pile deployment analysis and provides a complete workflow including:

- Data crawling
- Data storage
- Data analysis
- Machine learning
- Backend services
- Frontend visualization

This repository contains the complete source code, SQL scripts, analysis modules, and project documentation.

# Getting Started

## 1. Clone Repository

```bash
git clone https://github.com/xiao37103/smart-charging-pile-analysis-system.git
```

## 2. Import Database

Execute the following SQL files:

```text
样本数据.sql
sql.sql
```

## 3. Start Backend

Enter the backend directory:

```text
springboot-jishe/
```

Run the SpringBoot project using IntelliJ IDEA or Maven.

## 4. Start Frontend

Enter:

```text
前端代码/
```

Run the frontend project according to the project structure.

## 5. Run Analysis & Machine Learning Modules

Enter:

```text
数据挖掘/
机器学习/
```

Run the corresponding Python scripts and notebooks.
